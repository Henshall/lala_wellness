== Installation ==

### Using The WordPress Plugins Dashboard

1. Go to the 'Add New' in the plugins dashboard
2. Search for 'Meta Tags Optimization'
3. Click **Install** Now button
4. Then click **Activate Plugin** button
5. You can see the plugin installed on Wordpress left menu

### Usage

It's easy ! Once activated, check how meta tags are optimized from a edit posts or pages. Below description, a nice and easy to understand blocked "Meta Tags Optimization "available.
